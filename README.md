# IOT-AQI-Tracker

# 🌍 IoT Air Quality Monitoring Dashboard

A real-time air quality monitoring dashboard built using **Firebase**, **Leaflet.js**, and **JavaScript**.  
The system fetches IoT sensor data (Temperature, Humidity, Dust Density, AQI, etc.) from Firebase Realtime Database and visualizes it on an interactive map.

- **Firebase Web App:** [https://iot-airquality-tracker.web.app](https://iot-airquality-tracker.web.app)

- **GitHub Repository:** [https://github.com/Aniketh05/IOT-AQI-Tracker](https://github.com/Anikethb05/IOT-AQI-Tracker/)


---

## 🧠 Features

- 🌡️ Real-time sensor data updates  
- 🗺️ Interactive map (Leaflet.js) with color-coded AQI markers  
- 📊 Live readings of Temperature, Humidity, Dust Density, and Predicted AQI  
- ⚡ Firebase Hosting + Realtime Database integration  
- 🧭 Dynamic marker updates with AQI color indicators
